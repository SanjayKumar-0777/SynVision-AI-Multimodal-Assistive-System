# 🚀 SynVision – AI Multimodal Assistive System

## 🧠 Overview

**SynVision** is an AI-powered multimodal assistive system designed to enhance communication and accessibility for individuals with disabilities.
It integrates multiple intelligent modules such as **Sign Language Recognition** and **Face Recognition** using advanced **Computer Vision** and **Machine Learning** techniques.

---

## 🎯 Key Features

* ✋ Real-time **Sign Language Recognition** using hand gesture detection
* 👤 **Face Recognition System** for identity authentication
* 🔊 Assistive interaction through AI-driven modules
* ⚡ Modular architecture for scalability and integration
* 🌐 User-friendly interface with web-based interaction

## 🚀 Getting Started

### 🔧 Installation

```bash
git clone https://github.com/your-username/SynVision-AI-Multimodal-Assistive-System.git
cd SynVision-AI-Multimodal-Assistive-System
pip install -r requirements.txt
```

---

### ▶️ Run Sign Language Module

```bash
cd sign_language
python app.py
```

---

### ▶️ Run Face Recognition Module

```bash
cd face_recognize
python app.py
```
